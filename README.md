## 📌 Featured Projects

### 🧠 [LLM Q&A Application for Medical Data](https://github.com/keerthan-ugrani/LLM-Q-A-application-for-Medical-Data)
Applied LLM project leveraging **Retrieval-Augmented Generation (RAG)** to enable intelligent question-answering over medical datasets. Built end-to-end with embedding models, vector stores, and a Streamlit-based interface.

### 🔬 [Deep Learning for Class-Imbalanced Medical Image Classification](https://github.com/keerthan-ugrani/Praktikum)
Designed a robust deep learning framework to tackle class imbalance in medical imaging using resampling strategies, custom loss functions, and CNN architectures.

### 🎓 [Master's Thesis — Battery State of Health Estimation](https://github.com/keerthan-ugrani/Masters-Thesis)
End-to-end adaptive data analytics pipeline for battery health monitoring at Daimler Truck AG. Achieved **50% error reduction** in SOH estimation over 15TB+ of time-series data.

### 🧮 [Neural Networks & Sequence-to-Sequence Models](https://github.com/keerthan-ugrani/Neural-Networks-and-Sequence2Sequence)
Implementation and comparison of neural network architectures including sequence-to-sequence models for NLP tasks.
